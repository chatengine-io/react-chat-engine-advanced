# Chat Avatars

The `ChatAvatars` component appears at the top of the Settings window.

This is used to display the Avatars of the users in the chat.

Two arguments are required, the list of users in the chat, `users: PersonProps[]`, and the current user, `currentUser: PersonProps`.

Optionally set `isDirectChat = true` if the chat is a direct message. Otherwise this defaults to `false` and assumes a group chat.

## Direct Message Chat

For direct message chats, only the other person's avatar will display.

<Canvas>
  <Story
    name="Direct Message"
    args={{
      users: directMessageChat.people,
      currentUser: currentUser,
      isDirectChat: true,
    }}
  >
    {Template.bind({})}
  </Story>
</Canvas>

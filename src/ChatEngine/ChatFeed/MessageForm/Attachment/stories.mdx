# Attachment

This Attachment component represents a file you're attaching in a new message.

This `Attachment` component is slightly different than the sent message Attachment component.

Differences involve the ability to remove the attachment from a message, and sizing.

## Default

## Image URL

If you want to show an Image Thumbnail type attachment, then pass in a valid Image URL to `imageUrl` for the following style.

<Canvas>
  <Story
    name="Image URL"
    args={{
      fileName: 'my-face-min.png',
      imageUrl:
        'https://chat-engine-assets.s3.amazonaws.com/tutorials/my-face-min.png',
    }}
  >
    {Template.bind({})}
  </Story>
</Canvas>

## Custom Style

To customize the look and feel, use the `style` prop which has a `AttachmentStyle` interface.

# Chat Header

The Chat Header lives at the top of the Chat Feed and shows data about the Chat you have opened.

Things such as the chat title, members, and last sent message (or last message datetime) may be present.

## Default

<Story
  name="Default"
  args={{
    id: 1,
    title: 'Magic Club',
    description: 'Where the magic happens...',
  }}
>
  {Template.bind({})}
</Story>

## Custom Styles

You can customize the look and feel with the `style` prop which extends the `ChatHeaderStyles` interface.

# File

The File component represents any Message Attachment which cannot be put into an `<img />` tag.

## Default

<Story
  name="Default"
  args={{
    url: 'https://chat-engine-assets.s3.amazonaws.com/click.mp3',
  }}
>
  {Template.bind({})}
</Story>

<ArgsTable story="Default" />

## Is Loading

If you do not set a `url` prop, the image is considered loading. And will show the following loading text + emoji.

<Story name="Is Loading" args={{}}>
  {Template.bind({})}
</Story>

You can also specify the file name with the `fileName` prop and render your own content.

You can customize the look and feel with the `*Style` props and change the CSS for every Image state.

# Chat Avatars

The `ChatAvatars` component appears at the top of the Settings window.

This is used to display the Avatars of the users in the chat.

Two arguments are required, the chat object,`chat: ChatProps`, and the current user, `currentUser: PersonProps`.

## Direct Message Chat

For direct message chats, only the other person's avatar will display.

<Canvas>
  <Story
    name="Direct Message"
    args={{
      chat: directMessageChat,
      currentUser: currentUser,
    }}
  >
    {Template.bind({})}
  </Story>
</Canvas>

## One Person Chat

For groups chats containing one person, only their avatar will display.

  <Story
    name="One Person"

## Two Person Chat

For two person group chats, both avatars will display.

<Canvas>
  <Story

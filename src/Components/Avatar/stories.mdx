# Avatar

The Avatar visually represents a user.

Avatars's can also display a user's online status.

You can customize the look and feel with the `avatarUrl`, `username`, `showOnline`, `isOnline`, and `style` props.

## Default

<Story
  name="Default"
  args={{
    username: 'Adam',
  }}
>
  {Template.bind({})}
</Story>

<ArgsTable story="Default" />

## Is Online

When you set `showOnline` to `true`, then you can see the user's online status.

There will be a red dot if `isOnline` is `false`, and a green dot if `isOnline` is `true`.

<Canvas>
  <Story
    name="Is Online"
    args={{
      username: 'Adam',
      showOnline: true,
      isOnline: true,
    }}
  >
    {Template.bind({})}
  </Story>
</Canvas>

## Avatar Url

When you set the `avatarUrl` value, you can use a picture instead of `username`.

<Canvas>
  <Story
    name="No Avatar Url"
    args={{
      avatarUrl:
        'https://chat-engine-assets.s3.amazonaws.com/tutorials/my-face-min.png',
    }}
  >
    {Template.bind({})}
  </Story>
</Canvas>

## Custom Style

You can also use the `customStyle` component to overlay custom CSS onto the `Avatar` component.

<Canvas>
  <Story
    name="Custom Style"
    args={{
      username: 'Adam',
      isOnline: true,
      showOnline: true,
      style: { border: '2px solid red' },
      statusStyle: { border: '2px solid blue' },
    }}
  >
    {Template.bind({})}
  </Story>
</Canvas>

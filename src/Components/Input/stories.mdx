# Input

The `Input` component is where the user can add text to their message.

## Default

<Story name="Default" args={{}}>
  {Template.bind({})}
</Story>

<ArgsTable story="Default" />

## Functions

Pass in an `onChange` function to perform an action whenever the `value` of the component changes.

Use the `onFocus` function to perform an action when the user selects the input bar.

The `onBlur` function is used to perform an action when the user clicks out of the input bar.

## Label

Add a label using the `label` argument.

<Canvas>
  <Story name="Label" args={{ label: 'Send a message...' }}>

Get the user started using the `defaultValue` argument.
